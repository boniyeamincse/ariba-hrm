# Lab Technician Dashboard

## Menu and Submenu

- Lab Workbench
- Sample Management
  - Collected Samples
  - Pending Collection
- Test Processing
  - In Progress Tests
  - QC Validation
- Results
  - Result Entry
  - Critical Result Alerts

## Permission Role

- `lab.sample.manage`
- `lab.test.process`
- `lab.result.manage`
- `lab.alert.read`

## Capabilities

- Manage samples across collection lifecycle.
- Process and validate test runs.
- Publish results and raise critical alerts.
