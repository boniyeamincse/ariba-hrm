# Ward Manager Dashboard

## Menu and Submenu

- Ward Control Center
- Bed Management
  - Bed Matrix
  - Transfer Requests
- Staff Coordination
  - Shift Assignment
  - Task Load
- Clinical Supervision
  - Ward Round Summary
  - Nursing Compliance

## Permission Role

- `ward.dashboard.read`
- `ward.bed.manage`
- `ward.staff.manage`
- `ward.compliance.read`

## Capabilities

- Oversee bed allocation and transfer flow.
- Coordinate nursing/staff execution by shift.
- Track ward-level compliance and throughput.
