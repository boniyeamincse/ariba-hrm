# IT Admin / System Administrator Dashboard

## Menu and Submenu

- System Health
  - API Latency
  - Queue Health
  - Storage Utilization
- Access Security
  - Login Audit
  - MFA Status
  - Role Exceptions
- Integration Hub
  - API Clients
  - Webhook Delivery Logs
- Maintenance
  - Backup Status
  - Scheduled Jobs

## Permission Role

- `it.monitoring.read`
- `it.security.manage`
- `it.integration.manage`
- `it.maintenance.manage`

## Capabilities

- Monitor platform performance and security posture.
- Manage integrations, backups, and operational maintenance.
- Investigate incidents from logs and telemetry.
