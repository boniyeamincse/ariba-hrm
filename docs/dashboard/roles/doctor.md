# Doctor Dashboard

## Menu and Submenu

- My Schedule
  - Today Appointments
  - Follow-up Appointments
- OPD Queue
  - Current Token
  - Call Next
- Consultation
  - SOAP Notes
  - ICD-10 Search
- Prescription
  - E-Prescription Builder
  - Drug Interaction Alerts
- Investigations
  - Order Lab Tests
  - Order Radiology

## Permission Role

- `doctor.schedule.read`
- `opd.queue.call`
- `consultation.manage`
- `prescription.manage`
- `investigation.order`

## Capabilities

- Conduct consultations and finalize notes.
- Generate prescriptions with safety checks.
- Order investigations and track results.
