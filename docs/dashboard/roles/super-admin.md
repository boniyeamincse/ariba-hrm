# Super Admin Dashboard

## Menu and Submenu

- Platform Overview
- Tenant Management
  - Tenant List
  - Create Tenant
  - Subscription Plans
  - Usage Monitoring
- Global User Management
  - Role Templates
  - Permission Matrix
- System Monitoring
  - Service Health
  - Queue/Job Monitor
  - Error Logs
- Security and Compliance
  - Audit Trails
  - Access Review
- Integrations
  - API Keys
  - Webhooks

## Permission Role

- `platform.tenants.manage`
- `platform.roles.manage`
- `platform.permissions.manage`
- `platform.monitoring.read`
- `platform.security.read`
- `platform.integrations.manage`

## Capabilities

- Create, suspend, and reactivate tenants.
- Manage global role templates and defaults.
- View cross-tenant health dashboards.
- Configure platform integrations and API limits.
