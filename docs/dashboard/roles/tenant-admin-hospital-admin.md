# Tenant Admin (Hospital Admin) Dashboard

## Menu and Submenu

- Hospital Overview
- User and Role Management
  - Staff Accounts
  - Role Assignment
  - Access Policies
- Clinical Operations
  - OPD Snapshot
  - IPD Snapshot
  - Emergency Status
- Financial Control
  - Revenue Summary
  - Outstanding Bills
  - Insurance Claims
- Configuration
  - Hospital Settings
  - Department Settings
  - Notification Rules

## Permission Role

- `tenant.users.manage`
- `tenant.roles.assign`
- `tenant.settings.manage`
- `tenant.clinical.read`
- `tenant.finance.read`

## Capabilities

- Manage all hospital users in a tenant.
- View end-to-end clinical and financial performance.
- Control tenant-level settings and feature toggles.
