# Inventory Manager / Store Manager Dashboard

## Menu and Submenu

- Inventory Control
- Stock Status
  - Current Stock
  - Minimum Level Alerts
  - Near Expiry Items
- Procurement
  - Purchase Requests
  - Vendor Comparison
- Distribution
  - Department Requisitions
  - Issue/Return Log

## Permission Role

- `inventory.dashboard.read`
- `inventory.stock.manage`
- `inventory.procurement.manage`
- `inventory.distribution.manage`

## Capabilities

- Maintain stock balance and expiry safety.
- Run procurement cycle and vendor follow-up.
- Track internal issue/return movements.
