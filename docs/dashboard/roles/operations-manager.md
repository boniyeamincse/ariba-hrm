# Operations Manager Dashboard

## Menu and Submenu

- Daily Operations Board
- Queue Management
  - OPD Queue Status
  - Lab Queue Status
  - Pharmacy Queue Status
- Facility Operations
  - Room Turnaround
  - Ambulance Utilization
- SLA and Exceptions
  - Delayed Tasks
  - Incident Log

## Permission Role

- `ops.dashboard.read`
- `ops.queue.manage`
- `ops.facility.read`
- `ops.incident.manage`

## Capabilities

- Monitor real-time operational throughput.
- Resolve queue and service bottlenecks.
- Track SLA breaches and incidents.
