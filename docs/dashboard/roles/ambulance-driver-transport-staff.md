# Ambulance Driver / Transport Staff Dashboard

## Menu and Submenu

- Transport Board
- Dispatch Queue
  - New Dispatches
  - Active Trips
- Route and Status
  - Navigation Link
  - ETA Updates
- Handover
  - Patient Transfer Notes
  - Trip Completion

## Permission Role

- `transport.dispatch.read`
- `transport.trip.manage`
- `transport.status.update`

## Capabilities

- Receive and execute ambulance dispatch tasks.
- Update trip status and ETA in real time.
- Record transfer completion and handover notes.
