# Tenant Admin Dashboard

## Role: tenant-admin
## Email: admin@hospital.com
## Scope: Hospital Configuration & Full Data Access

### Primary Widgets
- Total Staff: Employee count in hospital
- Total Patients: Patient population
- Revenue Today: Daily hospital revenue
- Outstanding Bills: Unpaid invoices

### Quick Actions
1. Manage Users - Staff and user role management
2. OPD Queue - Outpatient department operations
3. Billing Console - Invoice and payment management
