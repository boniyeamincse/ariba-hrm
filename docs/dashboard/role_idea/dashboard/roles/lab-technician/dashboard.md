# Lab Technician Dashboard

## Role: lab-technician
## Email: lab@hospital.com
## Scope: Laboratory Order Fulfillment & Results

### Primary Widgets
- Lab Pending: Tests in progress
- Lab Completed Today: Completed tests
- Critical Lab Alerts: Abnormal results
- Open Tasks: Assigned lab tasks

### Quick Actions
1. Lab Queue: Order fulfillment
2. Result Entry: Enter test results
3. Sample Tracking: Specimen tracking

### Focus Areas
- Turnaround Time
- Critical Alerts
- Quality Assurance

### Menu Access
- Lab Orders
- Sample Management
- Result Entry
- Lab Reports
- Reference Ranges

### Access Level
- Lab order processing
- Sample collection recording
- Result entry and validation
- Report generation
- Quality control
