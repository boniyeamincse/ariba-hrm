# Super Admin Dashboard

## Role: super-admin
## Email: superadmin@medcore.com
## Scope: Global SaaS Operations (All Hospitals + All Data + All Settings)

### Primary Widgets
- Total Hospitals: Count of all tenant organizations
- Active Tenants: Count of operational hospitals
- Suspended Tenants: Count of inactive/suspended hospitals
- Global Users: Total users across all hospitals
- Revenue Today: Total payments received across all hospitals
- Payments Today: Total payment transactions processed
- API Calls Today: Platform integration usage
- Security Alerts: System security events requiring attention
- Webhook Failures: Failed integration webhooks
- Enabled Modules: Count of enabled feature modules

### Quick Actions
1. Tenant Management - Manage hospital organizations, activation, suspension
2. Global User Access - View/manage users across all hospitals
3. Subscription & Billing - Create plans, assign plans, monitor revenue
4. System Analytics - Cross-tenant KPIs and trends
5. Security & RBAC - Role permission policies, enforcement
6. API & Integrations - Third-party connectors, webhook configuration
7. Module Controls - Enable/disable features per hospital
8. Backup & Maintenance - System backups, database restore, maintenance mode
9. AI Governance - AI assistant usage, automation rules
10. Support Monitoring - Tenant support tickets, issue tracking

### Focus Areas
- Global SaaS Operations
- Tenant Lifecycle Management
- Subscription Revenue Oversight
- Security and Compliance Governance
- Integrations and Platform Reliability

### Access Level
- Full system access
- SaaS owner perspective
- No tenant restrictions
- Global reporting visibility
