# HR Manager Dashboard

## Role: hr-manager
## Email: hr@hospital.com
## Scope: Staff, Leave, Attendance & Payroll

### Primary Widgets
- Total Staff: Employee count
- Attendance Rate: Attendance percentage
- Open Tasks: HR tasks
- Active Sessions: System sessions

2. Attendance: Attendance tracking
3. Payroll: Payroll management
