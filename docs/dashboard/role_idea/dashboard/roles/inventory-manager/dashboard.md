# Inventory Manager Dashboard

## Role: inventory-manager
## Email: inventory@hospital.com
## Scope: Store, Assets, and Purchases

### Primary Widgets
- Low Stock Items: Below reorder level
- Open Tasks: Purchase orders and tasks
- Invoices Today: Vendor invoices
- Open Claims: Vendor disputes

2. Purchase Orders: PO management
3. Suppliers: Vendor management
